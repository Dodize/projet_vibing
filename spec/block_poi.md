# Etat actuel
Quand un utilisateur s'approche d'un POI, une popup apparait pour lui proposer d'accéder à l'interface de quiz.
Quand un POI a déjà été visité par un utilisateur, il n'est pas possible de cliquer à nouveau dessus dans la liste des POIs.

# Modification
Si le POI a déjà été visité par l'utilisateur le jour même, la popup ne doit pas apparaître. La méchanique est déjà présente pour la liste de POI, reprendre la même logique pour la popup.
